# MPPTSolarPwrModule_R2

MPPT Solar Power Module Revision 2

- 3.7V LiPo cell
- INA3221 Current/Voltage Monitor
    - Solar Panel Input
    - Battery Cell Output
    - Load Output 
- 6V/12V Solar Panel Selection
- Load Sharing of Solar Panel and Battery
- Regulated 6V output
